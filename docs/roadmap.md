# Roadmap

## v0.1 — Launch MVP (Esta Semana)

**Objetivo:** Publicar y empezar a compartir.

### Secciones

| Sección | Propósito |
|---|---|
| Hero | Value proposition + CTA |
| Services | Qué problemas resuelvo |
| About | Por qué confiar en mí |
| Featured Projects | Impacto, no tecnologías |
| Contact | Sin fricción |

### Calidad Mínima

- Lighthouse ≥ 90
- Dark mode
- Responsive (mobile-first)
- SEO básico (title, description, lang)
- TypeScript sin errores
- Link de contacto funcional (mailto + formulario)

### NO está en v0.1

- Blog, resume, casos de estudio, testimonios, analytics, SEO avanzado, animaciones elaboradas, tests automatizados

## v0.2 — Build Trust (Próximas Semanas)

**Objetivo:** Aumentar credibilidad después de tener feedback real.

Posibles adiciones (solo si responden a las 4 preguntas):

- Testimonios de clientes
- Casos de estudio con métricas
- Resume / CV descargable
- Mejora de SEO (JSON-LD, OG images)
- Mejora del formulario de contacto
- Animaciones sutiles

## v1.0 — Personal Brand (Futuro)

**Objetivo:** Autoridad a largo plazo.

Ejemplos:

- Blog técnico
- Página /uses, /now, /learning
- Auditoría de accesibilidad completa
- Analytics para entender tráfico
