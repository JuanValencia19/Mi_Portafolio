# Tech Stack

| Tecnología | Por qué | Alternativa |
|---|---|---|
| **Next.js 15** | App Router, Server Components, build optimizado | — |
| **TypeScript** | Evita errores en producción | — |
| **Tailwind CSS** | Rapidez, consistencia, dark mode nativo | CSS Modules (más lento) |
| **shadcn/ui** | Componentes accesibles listos para usar | Build from scratch (más lento) |
| **Framer Motion** | Animaciones simples que se ven profesionales | CSS animations (gratis, menos vistoso) |
| **react-hook-form + zod** | Formulario con validación en 10 líneas | Form nativo sin validación UX |
| **next-themes** | Dark mode sin flash | CSS custom (más trabajo) |
| **Vercel** | Deploy automático, SSL, dominio, 0 config | Netlify, Cloudflare Pages |

## Criterio de inclusión

¿Esta tecnología ayuda a que el sitio se vea profesional y convierta visitantes en leads?

Si la respuesta es "no" o "quizás en el futuro", no está en v0.1.
